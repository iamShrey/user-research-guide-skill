# Historical Interview Guides

This folder holds past interview guides used as reference when generating new ones.

Each guide should follow this structure so the skill can use it effectively:

---

## Filename format
`[topic-slug].md` — e.g. `onboarding-flow.md`, `checkout-usability.md`

## Required sections in each guide

```
# [Topic] Interview Guide

## Research Objective
[One paragraph describing what this research was trying to learn and why]

## Sub-Research Questions
1. [SRQ 1]
2. [SRQ 2]
...

## Target Participant
[Role, experience level, relevant qualifying behaviors]

## Session Length / Format

---
[The full interview guide sections follow]
```

---

Guides in this folder are loaded **selectively** — the skill picks the 1–2 most relevant to the current research objective, not all of them. Add as many as you like over time.

This folder is empty on first install. It gets populated either during the setup wizard (if you upload or paste existing guides) or manually over time.
